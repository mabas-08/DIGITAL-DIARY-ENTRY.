# Digital Diary Entry – A Day in My Life

This project is a web-based diary entry created for a Web Development assignment.

## Description
The website is only a front-end project about digital diaries. It has three pages with the home page showing an example of a digital diary, the second page explaining what is a digital diary and the last page with a form that enables you express your feelings. 
!caution!- as the website is build only on front-end it doesn't actually save your files.

## Features 
The colors and designs used are simple and calm to give some elegant vibe

## Technologies Used
The front-end technologies that are used on the website are:
- HTML
- CSS
