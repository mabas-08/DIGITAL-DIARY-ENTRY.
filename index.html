DIGITAL DIARY - A day in my life

Today was our first day of learning HTML-the basic markup language for developing a website. When we first started our web-design class it was really hard to keep up with the others as everything felt new and strange. By some time went on, the challenges kept on increasing but I developed the ability to overcome obstacles and learn something new everyday. Until this moment most of the class time was theory focused so it wasn't that much interesting. As today was the first practical class we started with the basic structures and elements of HTML like

  • the html element(which includes the whole html page)
  • the head element
  • the title element(inside the head element)
  • and also the body element
Our teacher introduced us to the interesting attributes of the elements like background color and also we get to know some important rules of writing HTML code.
In the past two days I tried to get some information about HTML and how it suppose to be written, so one of the difficulties was understanding why most of elements have start and end tags while some of them have only start tags but today I've came to understand the difference between normal elements and void ones. As a person who doesn't like tech, I found learning HTML fun and relaxing.
